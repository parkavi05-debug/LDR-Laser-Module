# 1️⃣ Smart Light Meter

### 📌 Description

Measures ambient light intensity using an **LDR sensor** and displays the light level on OLED.

### ⚡ Features

* Reads analog light value
* Detects Bright / Normal / Dark condition
* Displays data on OLED
